# documentation Git for beginners :

- to fetch the project use :

````bash
git clone git@github.com:zazoumdr/The-Binding-of-V1.git
````

make shure you have an ssh key connected to github first !

- to check the status of your files (modified, staged, untracked) :

```bash
git status
```

- to stage your changes :

```bash
git add .
```

- to commit your staged changes with a message :

```bash
git commit -m "your message here"
```

- to push your commits to github :

```bash
git push
```
